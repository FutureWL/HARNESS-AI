快速开始

开发模式：前端 Vite + Electron 主进程并行启动。生产模式：构建前端静态产物并打包客户端。

本地开发
pnpm install
pnpm dev
会同时启动 renderer 与 desktop（Electron 将加载 Vite 地址）。
构建 & 打包
pnpm build
pnpm package:dir
pnpm package:linux
产物在 release/，包含 linux-unpacked 与 AppImage。

下载与使用

AppImage 无需安装：赋予执行权限即可运行。在 Ubuntu 24.04 上如遇 FUSE 问题，可安装兼容库。

AppImage（Ubuntu 24.04）
文件Harness AI-0.1.0.AppImage

Nginx + Docker 门户
本页面支持容器化部署。把 portal/ 目录构建为 Nginx 静态站点即可。
查看部署

门户部署（Nginx + Docker）

适合用于项目介绍页、下载页与文档入口。零依赖：只需要 Docker 与 Docker Compose。

启动
cd portal
docker compose up -d
open http://localhost:33210
默认监听 33210，你可以在 compose 文件里修改端口。
更新内容
编辑 portal/site/index.html
编辑 portal/site/styles.css
docker compose restart
也可以把 portal/site 替换为你自己的静态站点产物。
